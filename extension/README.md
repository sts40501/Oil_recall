# E-Recall Chromium 擴充功能

此擴充功能不連線至第三方後端，也不需要財政部 API 憑證。

## 安裝

1. 在 Chrome／Edge 開啟擴充功能管理頁，啟用「開發人員模式」。
2. 點選「載入未封裝項目」，選取本資料夾 `extension/`。
3. 點擊工具列 E-Recall 圖示，會導向官方「發票查詢及捐贈」頁（`/portal/btc/mobile/btc502w`）。
4. 使用者自行登入、完成官方圖形驗證與月份查詢。
5. 查詢結果出現後，擴充功能會將每頁改為 100 筆、全選，並依序下載所有頁的 CSV；也可按右下角「下載所有頁 CSV」手動開始。
6. 點擊右下角「前往比對 CSV 檔案」，開啟 [E-Recall 網站](https://oilrecall.netlify.app/)；將下載的 CSV 拖曳至匯入區，在本機進行判讀。

## 隱私與限制

- 擴充功能只要求 `www.einvoice.nat.gov.tw` 的頁面權限。
- 不讀取輸入的帳號、密碼、驗證碼或 cookie，不保存任何登入狀態。
- 不代填登入資料、不自動點擊或繞過圖形驗證，也不替使用者決定查詢月份。
- 只在使用者點擊工具列圖示授權後，自動操作已顯示的查詢結果頁來選取與下載 CSV。
- CSV 只下載到使用者裝置；擴充功能不會自動上傳檔案或傳送發票資料。

官方網站版面若改動，可能影響可辨識的資訊；遇到無法比對時，請改用網站版 CSV／QR Code 手動比對。
