# E-Recall 電子發票毒油回收快速查詢系統

協助台灣消費者以本機方式比對中聯油脂案受影響商品、業者與發票資訊。

## 功能

- 網站版：可一次匯入多個月份 CSV、電子發票 QR Code 掃描／貼上、手動關鍵字比對、流向視覺化與完整清單瀏覽。
- Chromium 擴充功能：使用者自行登入官方電子發票平台並開啟發票明細後，點擊擴充功能即可在**本機**比對目前頁面文字。
- 內建資料：440 項預防性下架產品、489 筆食藥署下游業者資料，以及泰山 115/07/12 公開的 4,691 筆流向紀錄。

本專案不使用、不申請，也不保存財政部 AppID、APIKey、載具密碼或網站 cookie。

## 發票判讀原則

- 明確、具辨識度的完整品名才可直接列為公告候選，並提示使用者核對效期或批號。
- 「熟食、麵包、餐飲、沙拉油」等通用品名，必須同時符合公告店家才會提示。
- 店家名稱曾出現在流向清單，不代表該店所有商品受影響；店家名稱不會單獨觸發商品警示。
- 「未命中清單」只代表目前公告資料沒有相符品項，不等同食品安全保證。

## 資料與限制

官方查詢頁要求使用者自行完成登入與圖形驗證，且每次查詢區間限同一月份。擴充功能不代填登入資料、不繞過驗證碼、不自動發送官方查詢；它只讀取使用者目前已登入且可見的發票頁面並在瀏覽器本機比對。[官方載具查詢說明](https://www.einvoice.nat.gov.tw/portal/btc/vehicle/btc201w/search)

## 免責聲明

資料彙整自食藥署公告（截至 115/07/09）及泰山企業 115/07/12 公開流向附件。流向紀錄不等同個別消費品已檢出不合格；請以食藥署、業者公告與商品批號為準。
